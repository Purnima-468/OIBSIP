# 🌡️ Temperature Converter

## 📌 Project Overview
The Temperature Converter is a simple and responsive web application built using HTML, CSS, and JavaScript. It allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin with real-time calculations and input validation.

## 🚀 Features
- Convert temperature between Celsius, Fahrenheit, and Kelvin.
- User-friendly interface.
- Input validation to prevent empty or invalid values.
- Absolute zero temperature validation.
- Responsive design for desktop and mobile devices.
- Fast and accurate conversions.

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## ▶️ How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.
4. Enter a temperature value.
5. Select the input unit (Celsius, Fahrenheit, or Kelvin).
6. Click the **Convert** button to view the converted temperatures.

## 📖 Temperature Conversion Formulas
- Celsius → Fahrenheit: `(°C × 9/5) + 32`
- Fahrenheit → Celsius: `(°F − 32) × 5/9`
- Celsius → Kelvin: `°C + 273.15`
- Kelvin → Celsius: `K − 273.15`
